更新ruby源报错bad response Not Found 404

Posted on 2018-12-06

添加国内ruby源 gem sources –add https://gems.ruby-china.org 报错： Error fetching https://gems.ruby-china.org/: bad response Not Found 404 (https://gems.ruby-china.org/specs.4.8.gz)

添加国内ruby源 gem sources –add https://gems.ruby-china.org

报错：

Error fetching https://gems.ruby-china.org/:

bad response Not Found 404 (https://gems.ruby-china.org/specs.4.8.gz)

原因是 ruby-china 更换了域名

https://gems.ruby-china.com

命令替换为 gem sources –add https://gems.ruby-china.com 解决
